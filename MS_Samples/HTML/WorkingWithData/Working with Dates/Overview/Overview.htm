DateTime values can be:

  • Bound to the values of data points.
  • Used as the interval type for axes and their associated grid lines, tick marks and strip lines.
  • Assigned to properties at design-time in the Visual Studio designer.
  • Formatted using standard or custom .NET date/time formatting.

Data Point Values

The X values of data points can be bound to a data source that has DateTime values.
